# SRS v1.1 — Sistema API Inventario

> Estado: Baselined | Versión doc: 1.1 | Aprobado en: v1.1.0

---

## Requisitos Funcionales

**REQ-001:** El sistema permitirá listar productos.  
**REQ-002:** El sistema permitirá agregar productos con cantidad >= 0.  
**REQ-003:** Filtrar productos por fecha (criterios a definir en ISSUE-21 — pendiente de implementación completa).  

## Requisitos No Funcionales

**RNF-001:** Los cambios deben ser trazables a un ISSUE y evidencias.  
**RNF-002:** Versionado seguirá SemVer con tags y changelog.  
**RNF-003:** No se almacenarán secretos ni credenciales en el repositorio.
